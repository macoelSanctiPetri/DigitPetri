# DigitPetri

Plan estratégico de digitalización del alumnado de 1º a 4º de ESO en un centro de Educación Secundaria. Se basa en la propuesta actual del centro y la enlaza con el marco europeo de competencia digital DigComp 3.0.

## Objetivos
- Definir líneas de acción para la digitalización del alumnado de 1º a 4º de ESO.
- Alinear la propuesta del centro con el marco europeo DigComp 3.0.
- Servir como documento base para el desarrollo de acciones y recursos.
